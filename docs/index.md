# Gescharについて

Geschar(ジェスチャー)は、5ch掲示板を効率的に閲覧するための専用ブラウザです。
2023年7月の5chの仕様変更により、誰でも自由に専用ブラウザを開発することができるようになったことを受け、翌8月に開発がスタートし、8月31日にバージョン1.0.0がリリースされました。

現時点ではまだ機能が少なく、動作が不安定ですが、今後アップデートを重ねてより良いアプリにしていきたいと考えています。

  - 再現性がなく、原因の特定に難航しています。

## 既知の不具合

- ジェスチャの表示が画面に残り続ける
  - 改善予定です。未定義のコマンドを適当に入力して消すことができます。
- 画面遷移直後にジェスチャが暴発する
  - 特にスレッド画面から板画面に戻った直後にジェスチャの反応が悪くなり、意図しないジェスチャコマンドが入力される現象です。  
- 書き込めない
- レスのポップアップやリンクが開かなくなる
  - 再現性がなく難儀しています。レスメニューなどを1度表示すると直ることが多いです。 
- 全体的に動作がもっさりしている
  - 1つ1つ原因を特定しながら改善中です。Ver.1.2.0で多少改善されました。
- スレッドのスクロール位置が安定しない(更新のたびに少しずれる)
  - SwiftUIでとても苦労している部分です。改善に向けて努力します。
- リンクをタップしてWebViewを開いても何も表示されない事がある
  - 回避策として、右下のSafariアイコンからブラウザで表示できます。
- IDの末尾にBEの情報がつながっている
- ジェスチャのカスタマイズを行っている場合に、アップデートで新しいジェスチャが追加された時、その新しいジェスチャはコマンド無登録状態となっている
- 他多数

- iPadの2カラム表示で、板画面のスクロールやジェスチャの反応が非常に悪くなりカクカクする

iPadの2カラム画面は考慮する事が多く、またどうしてもiPhoneでの動作確認が優先となってしまうため、特にバグが多いです。少しずつ修正しますので何卒ご了承ください。とりあえずの回避策として、Split ViewやSlide Overで1カラムで使用すると、iPhoneと同じ動作になります。

## Tips

- 現在の仕様では、レスのポップアップを閉じる際にフェードアウトのアニメーションがかかり、完全に消えるまでスレッドのスクロールが効かなくなります。しかし、ポップアップが表示された状態のままスレッドをスクロールすると、レスのポップアップが消え始めると同時にスクロールも実行され、少し快適です。※これができるのは、ポップアップが1つだけ表示されている状態に限ります。
- 現在の仕様では、レスのポップアップを一括で閉じる方法がなく、たくさん開いた時に閉じるのが大変です。しかし、SwiftUIの奇妙な動作として、Appスイッチャーを表示すると、たくさん開いていたポップアップが勝手に閉じ、2つだけ残ります。これは意図した動作ではありませんが、たくさん開いたポップアップをすぐに閉じる回避策として使用できます。※デバイスやiOSバージョンによって、動作が異なる可能性があります。※Appスイッチャーは、以下の方法で表示できます。
  - Face IDを搭載したiPhoneの場合: 画面の下部から上にスワイプしてから、中央で止めます。
  - ホームボタンを搭載したiPhoneの場合: ホームボタンをダブルクリックします。
- レスのポップアップが下に大きく開いて欲しい状況で、上に小さく開くことがあります。このとき、一度閉じて再度開くと下に大きく開いてくれる場合があります。
- スレッド画面の検索ウィンドウでは、検索ワードが未入力の時、全てのレスが表示されます。また、検索ウィンドウ内でも上下に移動するジェスチャを使うことができます。
  - この特性を利用して、スレッドの途中で最初の方のレスを一時的に確認するような使い方ができます。
  - 赤レス抽出の最低被レス数を0にすることでも同様のことができます。
- 一行だけのテキスト入力欄(検索欄など)では共通して、「改行」キーでキーボードを閉じることができます。
  - 本来はキーボードを閉じるボタンを用意するべきですが、現在実装には至っていません。

## Q&A

- 初期状態で板のカテゴリが少ない
  - 設定タブ > コンテンツに関する設定 > コンテンツの設定 > 「表示するカテゴリ」フィールドよりカテゴリを追加できます。
  - 行頭に「全て」と入力すると、カテゴリを一括で追加することもできます。
  - 入力後、下のボタンより板一覧を更新してください。
- レスセルの左側の3分の1をタップすると何が起こる？
  - そのレスに対して返信したレスを表示します。誰も返信していない場合は(現時点では)何も起こりません。
  - アンカーやリンクは、左側3分の1の範囲上にあっても、優先的にタップできます。
- 板画面の勢いアイコンについて
  - 勢い5,000未満は白抜き、5,000以上は塗りつぶしの炎アイコンになります。10,000以上は色がつきます。
- 勢いが3,000と9,900のものをよく見かけるのは？
  - レス数が少ないスレッドでは、勢いの上限を3,000または9,900に設定しています(序盤に無意味な書き込みで意図的に勢いを上げる行為への対策)。
- ↓と↑で始まるジェスチャコマンドがないのは？
  - 上下のドラッグ操作はスクロールに使われるため、ジェスチャコマンドは必ず←か→で始まります。
- 板選択画面の歩いている人のマークの意味は？
  - その板に訪れたことがあることを表します。24時間以内に訪れた板にはマークに色がつきます。
